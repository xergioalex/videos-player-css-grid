### How to make updates

Run local server:
```
yarn server
```

Run sass watcher:
```
yarn sass:watcher
```

**Optional:** Open the project using vscode docker container.

Run setup script:
```
cd docker/local
bash docker.sh base
```

Reset folder project permissions:
```
chown -R user_id:group_id movies-page-css-grid
```
